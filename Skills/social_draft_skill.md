# Skill: Social Media Draft (Cloud Agent)

**Domain:** Social Media
**Agent Type:** Cloud
**Capability:** Draft social media posts (Facebook, Instagram, X, LinkedIn)
**Execution:** Local Agent (after approval)

---

## Overview

This skill enables the Cloud Agent to:
1. Monitor `/Needs_Action/social/` for social media requests
2. Generate draft posts for multiple platforms
3. Write drafts to `/Pending_Approval/social/`
4. Signal Local Agent for approval

**CRITICAL:** Cloud Agent CANNOT publish posts. All posts are draft-only and require Local Agent approval.

---

## Usage

### As Cloud Agent

```python
from claim_task import TaskClaimer

# Initialize
claimer = TaskClaimer(vault_path='/opt/platinum-vault')

# Scan for social tasks
social_tasks = claimer.list_available_tasks(domain='social')

for task_file in social_tasks:
    # Claim task atomically
    success, message = claimer.claim(task_file, 'cloud')
    
    if success:
        # Read request content
        task_path = Path(vault_path) / 'Needs_Action' / task_file
        content = task_path.read_text()
        
        # Generate draft posts for each platform
        drafts = generate_social_drafts(content)
        
        # Write to Pending_Approval
        task_id = generate_task_id(content)
        draft_path = Path(vault_path) / 'Pending_Approval' / 'social' / f'{task_id}.md'
        draft_path.write_text(drafts)
        
        # Write status update
        claimer.write_status_update(task_id, 'draft_created', f'Social draft ready: {draft_path}')
        
        # Signal Local Agent
        claimer.write_signal('approval_needed', f'Social draft ready: {task_id}')
```

## Platform Guidelines

### Facebook
- Max characters: 63,206
- Best practices: Include image, keep text concise
- Posting: Via Facebook Graph API

### Instagram
- Caption max: 2,200 characters
- Hashtags: Up to 30
- Posting: Via Instagram Graph API

### X (Twitter)
- Max characters: 280 (4,096 for Premium)
- Hashtags: 1-3 recommended
- Posting: Via Twitter API v2

### LinkedIn
- Max characters: 3,000
- Best practices: Professional tone, include article links
- Posting: Via LinkedIn API

---

## Security Rules

**Cloud Agent MUST:**
- ✅ Read social requests from `/Needs_Action/social/`
- ✅ Generate draft posts
- ✅ Write drafts to `/Pending_Approval/social/`
- ✅ Write status to `/Updates/`
- ✅ Signal Local Agent when approval needed

**Cloud Agent MUST NOT:**
- ❌ Access social media credentials
- ❌ Publish posts directly
- ❌ Access WhatsApp sessions
- ❌ Access banking credentials
- ❌ Write to Dashboard.md
