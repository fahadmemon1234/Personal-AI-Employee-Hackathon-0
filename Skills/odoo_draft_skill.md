# Skill: Odoo Draft (Cloud Agent)

**Domain:** Accounting
**Agent Type:** Cloud
**Capability:** Create draft Odoo invoices and journal entries
**Execution:** Local Agent (after approval)

---

## Overview

This skill enables the Cloud Agent to:
1. Monitor `/Needs_Action/accounting/` for accounting requests
2. Query Odoo for partner/balance information (read-only)
3. Generate draft invoices/journal entries
4. Write drafts to `/Pending_Approval/accounting/`
5. Signal Local Agent for approval

**CRITICAL:** Cloud Agent CANNOT post to Odoo. All entries are draft-only and require Local Agent approval.

---

## Usage

### As Cloud Agent

```python
from claim_task import TaskClaimer
import requests

# Initialize
claimer = TaskClaimer(vault_path='/opt/platinum-vault')
odoo_url = os.getenv('ODOO_URL', 'http://localhost:8069')

# Scan for accounting tasks
accounting_tasks = claimer.list_available_tasks(domain='accounting')

for task_file in accounting_tasks:
    # Claim task atomically
    success, message = claimer.claim(task_file, 'cloud')
    
    if success:
        # Read request content
        task_path = Path(vault_path) / 'Needs_Action' / task_file
        content = task_path.read_text()
        
        # Generate draft entry
        draft = generate_odoo_draft(content)
        
        # Write to Pending_Approval
        task_id = generate_task_id(content)
        draft_path = Path(vault_path) / 'Pending_Approval' / 'accounting' / f'{task_id}.md'
        draft_path.write_text(draft)
        
        # Write status update
        claimer.write_status_update(task_id, 'draft_created', f'Odoo draft ready: {draft_path}')
```

def search_odoo_partners(query: str) -> list:
    """Search for partners in Odoo (read-only)."""
    response = requests.post(f'{odoo_url}/jsonrpc', json={
        'jsonrpc': '2.0',
        'method': 'call',
        'params': {
            'service': 'object',
            'method': 'execute_kw',
            'args': [
                os.getenv('ODOO_DATABASE'),
                os.getenv('ODOO_USERNAME'),
                os.getenv('ODOO_PASSWORD'),
                'res.partner',
                'search_read',
                [[['name', 'ilike', query]]],
                {'fields': ['name', 'email', 'phone'], 'limit': 10}
            ]
        },
        'id': 1

## Odoo MCP Integration

### Cloud Mode (Draft-Only)

Cloud Agent's Odoo MCP runs in draft mode:

```python
# mcp_odoo_server.py --mode draft

@app.route('/tools/draft_invoice', methods=['POST'])
def draft_invoice():
    """Create draft invoice (Cloud Agent - draft only)."""
    data = request.json
    
    # Generate draft structure
    draft = {
        'move_type': 'out_invoice',
        'partner_id': data.get('partner_id'),
        'invoice_line_ids': data.get('lines', []),
        'status': 'draft'  # Not posted
    }
    
    return jsonify({
        'success': True,
        'draft': draft,
        'message': 'Draft created. Requires Local approval to post.'
    })
```

### Local Mode (Full Execution)

Local Agent's Odoo MCP runs in full mode:

```python
# mcp_odoo_server.py --mode full

@app.route('/tools/create_invoice', methods=['POST'])
def create_invoice():
    """Create and post invoice (Local Agent - full access)."""
    data = request.json
    
    # Create invoice in Odoo
    result = odoo_api.execute_kw(
        db, uid, password,
        'account.move',
        'create',
        [data]
    )
    
    # Post invoice
    odoo_api.execute_kw(
        db, uid, password,
        'account.move',
        'action_post',
        [[result]]
    )
    
    return jsonify({
        'success': True,
        'invoice_id': result,
        'message': 'Invoice created and posted'
    })
```

## Security Rules

**Cloud Agent MUST:**
- ✅ Read accounting requests from `/Needs_Action/accounting/`
- ✅ Query Odoo for read-only data (partners, balances)
- ✅ Generate draft invoices/entries
- ✅ Write drafts to `/Pending_Approval/accounting/`
- ✅ Write status to `/Updates/`

**Cloud Agent MUST NOT:**
- ❌ Post invoices to Odoo
- ❌ Post journal entries to Odoo
- ❌ Modify Odoo data
- ❌ Access Local Agent credentials
- ❌ Write to Dashboard.md
