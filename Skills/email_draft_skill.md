# Skill: Email Draft (Cloud Agent)

**Domain:** Email
**Agent Type:** Cloud
**Capability:** Triage and draft email replies (DRAFT ONLY)
**Execution:** Local Agent (after approval)

---

## Overview

This skill enables the Cloud Agent to:
1. Monitor `/Needs_Action/email/` for new email tasks
2. Read and triage incoming emails
3. Generate draft replies using AI
4. Write drafts to `/Pending_Approval/email/`
5. Signal Local Agent for approval

**CRITICAL:** Cloud Agent CANNOT send emails. All replies are draft-only and require Local Agent approval.

---

## Usage

### As Cloud Agent

```python
from claim_task import TaskClaimer

# Initialize
claimer = TaskClaimer(vault_path='/opt/platinum-vault')

# Scan for email tasks
email_tasks = claimer.list_available_tasks(domain='email')

for task_file in email_tasks:
    # Claim task atomically
    success, message = claimer.claim(task_file, 'cloud')
    
    if success:
        # Read email content
        task_path = Path(vault_path) / 'Needs_Action' / task_file
        content = task_path.read_text()
        
        # Generate draft (using Claude Code or LLM)
        draft = generate_email_draft(content)
        
        # Write to Pending_Approval
        task_id = generate_task_id(content)
        draft_path = Path(vault_path) / 'Pending_Approval' / 'email' / f'{task_id}.md'
        draft_path.write_text(draft)
        
        # Write status update
        claimer.write_status_update(task_id, 'draft_created', f'Draft ready: {draft_path}')
        
        # Signal Local Agent
        claimer.write_signal('approval_needed', f'Email draft ready: {task_id}')
```

## Security Rules

**Cloud Agent MUST:**
- ✅ Read emails from `/Needs_Action/email/`
- ✅ Generate draft replies
- ✅ Write drafts to `/Pending_Approval/email/`
- ✅ Write status to `/Updates/`
- ✅ Signal Local Agent when approval needed

**Cloud Agent MUST NOT:**
- ❌ Access Gmail credentials
- ❌ Send emails directly
- ❌ Access WhatsApp sessions
- ❌ Access banking credentials
- ❌ Write to Dashboard.md
